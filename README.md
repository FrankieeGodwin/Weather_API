Weather App
A responsive weather application built with React that fetches real-time weather data from the OneWeather API. Users can search for any city and view current temperature, weather conditions, humidity, and wind speed in a clean, minimal interface.

Features
Search for weather information by city name

Real-time data from the OneWeather API

Displays temperature, weather condition, humidity, and wind speed

Fully responsive design for various screen sizes

Modular and reusable React components

Tech Stack
Frontend: React, CSS

API: OneWeather API
